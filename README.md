
# AgroCalc

Ferramenta para cálculo da necessidade de calcário em cana-de-açúcar (fórmula tradicional).

## Estrutura
- `index.html`: página principal
- `css/style.css`: estilos
- `js/script.js`: cálculo e lógica
- `img/`: imagens ilustrativas

## Como usar localmente
1. Extraia o arquivo ZIP.
2. Clique em `index.html` para abrir no navegador.

## Como hospedar no GitHub Pages
1. Crie um repositório no GitHub (ex: AgroCalc).
2. Faça upload de todos os arquivos.
3. Vá em **Settings → Pages → Source → main / root**.
4. Clique em **Save**. O link do site será: `https://seuusuario.github.io/AgroCalc`.
